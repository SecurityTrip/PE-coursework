## Работа с репозиторием

### 1. Клонирование репозитория
```bash
git clone --recurse-submodules https://github.com/SecurityTrip/PE-coursework.git
```
Если уже склонировали без субмодулей, выполните:
```bash
git submodule init
```
```bash
git submodule update
```
